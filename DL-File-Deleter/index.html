Moodle-Files-Deleter






Guide

IMPORTANT: delete files and folders separately!

Note: If there are a lot of files or folders being deleted, enter an amount less than the total number of files/folders. In other words, don't enter an amount greater than the number of files/folders you want to delete or else the application will crash.

Enter course url, username, password, and either # folders / # files (only one!)

ex: Fill out # folders and click run. Once this finishes, make sure to click save in the browser. Clear your entry for # folders, and then you can enter # files to delete and click run

Once the browser has opened up the section with all the files and folders, it will take about 10 seconds for the files to start getting deleted.

The browser will automatically close after 2 minutes so make sure to hit save

If something breaks just restart the application :)
